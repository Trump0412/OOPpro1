课程成绩计算说明：
1. 该门课程分为4个子课程，分别是数理逻辑、集合论与抽象代数、图论，课程总成绩=4门子课程学习情况成绩+平时综合+期中测试+小论文+期末考试*50%。
2. 4门子课程每门课学习情况各5分，满分合计20分；四门课中，各等级分为：A：5分；B：4分；C：3分；D：1分，其它为0分；忽略等级+，-；。
3. 平时综合、期中测试、小论文满分均为10分，其中，A：9.5分；B：8分；C：6分；D：3分，其它0分；若等级有+或-，则在相应等级分上加0.5分或减0.5分。
提示：请将上述4门子课程与平时综合、期中测试、小论文与期末考试均作为课程来处理，最后得到该学生的总成绩即可，所有操作（增删改查、统计分析等）均可针对包括总成绩在内的上述9项内容（总成绩由计算得到，不能直接修改）。
